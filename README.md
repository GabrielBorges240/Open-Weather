# Open Weather

Aplicação que exibe o clima atual consumindo a API OpenWeather.

Demo:  https://gabrielborges240.github.io/Open-Weather/

## Funcionalidades
- Busca por cidade
- Exibição de temperatura e clima
- Tratamento de erro da API

## Tecnologias
- HTML5
- CSS3
- JavaScript
- OpenWeather API

## Como executar
1. Insira sua chave da API no código
2. Abra o arquivo `index.html` no navegador
